# 占卜应用打包到手机说明

## 方法一：使用 Buildozer（推荐用于 Android）

### 1. 准备工作

你需要安装以下环境：

**Windows 用户（推荐使用 WSL）：**
```bash
# 启用 WSL（Windows Subsystem for Linux）
# 在 Windows 功能中启用"适用于 Linux 的 Windows 子系统"
# 然后从 Microsoft Store 安装 Ubuntu
```

**在 WSL Ubuntu 中安装依赖：**
```bash
# 更新系统
sudo apt update
sudo apt upgrade -y

# 安装必要的工具
sudo apt install -y git python3-pip build-essential \
    zlib1g-dev libncurses5-dev libncursesw5-dev \
    libtinfo5 cmake libffi-dev libssl-dev

# 安装 Buildozer
pip3 install --user buildozer

# 安装 Cython
pip3 install --user cython
```

### 2. 准备项目文件

确保你的项目目录包含以下文件：
- `divination_app.py`（主程序，需要重命名为 main.py）
- `trigrams.py`
- `dayan_divination.py`
- `number_divination.py`
- `buildozer.spec`（已创建）

**重要：** 将 `divination_app.py` 重命名为 `main.py`！

```bash
cp divination_app.py main.py
```

### 3. 开始打包

在项目目录下运行：

```bash
# 第一次运行会下载 Android SDK、NDK 等，需要较长时间
buildozer android debug
```

打包完成后，APK 文件会在 `bin/` 目录中。

### 4. 安装到手机

1. 用 USB 连接手机
2. 确保手机开启了"开发者选项"和"USB 调试"
3. 运行以下命令安装：

```bash
adb install bin/你的apk文件名.apk
```

或者直接将 APK 文件复制到手机中安装。

## 方法二：使用 Kivy Launcher（快速测试，最简单）

### 1. 安装 Kivy Launcher

在 Google Play Store 搜索并安装 "Kivy Launcher"。

### 2. 准备文件

将以下文件复制到手机的 `/sdcard/kivy/` 目录：
- `main.py`（重命名后的主程序）
- `trigrams.py`
- `dayan_divination.py`
- `number_divination.py`

### 3. 运行应用

打开 Kivy Launcher，找到你的应用并运行！

**注意：** Kivy Launcher 主要用于测试，不适合正式发布。

## 方法三：使用 Python for Android

如果 Buildozer 遇到问题，可以尝试直接使用 python-for-android：

```bash
# 安装 python-for-android
pip install python-for-android

# 构建 APK
p4a apk --package=org.example.divination \
    --name="占卜起卦" \
    --version=1.0 \
    --bootstrap=sdl2 \
    --requirements=python3,kivy \
    --window \
    --orientation=portrait \
    --main=main.py
```

## 常见问题

### 问题 1：Buildozer 下载慢

解决方案：
```bash
# 使用国内镜像源
# 修改 ~/.buildozer/buildozer.spec 中的下载地址
```

### 问题 2：找不到 buildozer 命令

解决方案：
```bash
# 将用户本地 bin 目录添加到 PATH
echo 'export PATH=$PATH:~/.local/bin' >> ~/.bashrc
source ~/.bashrc
```

### 问题 3：打包失败

常见原因：
- 内存不足（至少需要 4GB 可用内存）
- 磁盘空间不足（至少 10GB）
- 网络问题导致下载失败
